# Styling với Tailwind CSS

Dự án này sử dụng **Tailwind CSS v4** để styling. Tất cả styles đã được cấu hình sẵn trong Storybook.

## Cách sử dụng

### Utility Classes
Sử dụng các utility classes của Tailwind trực tiếp trong components:

```tsx
<div className="flex items-center gap-4 p-6 bg-white rounded-lg shadow-md">
  <Button className="bg-blue-500 hover:bg-blue-600 text-white">
    Click me
  </Button>
</div>
```

### Responsive Design
Tailwind cung cấp responsive modifiers:

```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  {/* Content */}
</div>
```

### Dark Mode
Sử dụng `dark:` modifier cho dark mode:

```tsx
<div className="bg-white dark:bg-gray-900 text-gray-900 dark:text-white">
  {/* Content */}
</div>
```

## Utility Functions

### cn() - Class Name Merger
Dự án sử dụng `cn()` utility để merge class names:

```tsx
import { cn } from '@/lib/utils'

function Component({ className }: { className?: string }) {
  return (
    <div className={cn('base-classes', className)}>
      {/* Content */}
    </div>
  )
}
```

### Class Variance Authority (CVA)
Sử dụng CVA cho variants phức tạp:

```tsx
import { cva } from 'class-variance-authority'

const buttonVariants = cva(
  'base-classes',
  {
    variants: {
      variant: {
        default: 'bg-primary text-white',
        outline: 'border border-gray-300',
      },
      size: {
        sm: 'px-3 py-1 text-sm',
        lg: 'px-6 py-3 text-lg',
      },
    },
  }
)
```

## Design Tokens

### Colors
- **Primary**: Blue shades
- **Secondary**: Gray shades
- **Destructive**: Red shades
- **Muted**: Subtle grays

### Spacing
- `gap-{n}`: Gap between flex/grid items
- `p-{n}`: Padding
- `m-{n}`: Margin

### Border Radius
- `rounded-sm`: 0.125rem
- `rounded`: 0.25rem
- `rounded-md`: 0.375rem
- `rounded-lg`: 0.5rem
- `rounded-full`: 9999px

## Best Practices

### 1. Consistent Spacing
Sử dụng spacing scale của Tailwind (4, 8, 12, 16, 24, 32, etc.)

### 2. Semantic Colors
Sử dụng semantic color names thay vì hardcoded colors:
- ✅ `text-primary` 
- ❌ `text-blue-500`

### 3. Reusable Components
Tạo components tái sử dụng thay vì lặp lại classes:

```tsx
// ✅ Good
<Button variant="primary">Click</Button>

// ❌ Avoid
<button className="px-4 py-2 bg-blue-500 rounded...">Click</button>
```

### 4. Responsive First
Thiết kế mobile-first, sau đó thêm breakpoints:

```tsx
<div className="text-sm md:text-base lg:text-lg">
  Responsive text
</div>
```
